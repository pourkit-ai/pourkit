# Architect Commands

Use short commands. The skill owns the workflow.

```txt
Architect: init <initiative title>
Architect: compress
Architect: status
Architect: next
Architect: publish PRD
Architect: breakdown
Architect: reconcile
Architect: update
Architect: list
```

Generic examples:

